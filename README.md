# Ice skating quiz

A simple quiz game for learning very specific ice skating rules for a very specific ice skating exam.

Based on [ISU Communication No. 2701 (2025/26)](https://www.isu.org/isu-communications/?tab=ISU%20Communications).

## Features

- Quiz for pair skating element penalties
- Quiz for pair skating GOE (grade of element) plus bullets
- Automatic check for new software updates
